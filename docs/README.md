# 📚 Documentação do Projeto NovaCore's

Esta pasta contém documentação relacionada ao projeto, guias de design e referências.

## 📄 Arquivos

- **DESIGN_GUIDE.md** — Guia completo de design do projeto (cores, tipografia, animações, etc)

## 🎨 Informações Rápidas

- **Tema**: Dark mode com acentos ciano e roxo (Cyberpunk Minimalism)
- **Fontes**: Syne (títulos) e DM Sans (corpo)
- **Cores Principais**:
  - Fundo: #0A0A0F (preto profundo)
  - Accent: #00D4FF (ciano) + #6C63FF (roxo)
  - Texto: #F0F0F5 (branco morno)
